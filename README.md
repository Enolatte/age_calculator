# 測試月齡計算工具

此工具用於計算測驗日期與出生日期的月齡，並考慮早產矯正年齡。

## 使用方法
1. 輸入測驗日期（格式：yyyy/mm/dd）。
2. 輸入受測者出生日期（格式：yyyy/mm/dd）。
3. 若受測者小於 2 歲，需輸入出生週數和預產期。
